  This is a full fledged pharmacy POS software which is developed using Python and by it's  library known as Tkinter for front-end purpose. Which also provides easy and fast interaction to the user. It is also connected to a MySQL database for storing and retrieving data when needed and asked.      
By using this POS software user can handle all the possible operations which can be performed in a pharmacy. Whether it is record  keeping, Warehouse management, Employee Management, Sale Management and Generating bill and tracking them by using its bill no. as well. Also provides separate working space for admin and employee vice versa.
